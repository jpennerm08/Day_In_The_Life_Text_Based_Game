# Day In The Life

Day In The Life is an example of a Python text-based adventure game.
The goal of the game is to make the best decision with the goal of getting a promotion at your job.
Your boss hates you so this isn't as easy as you think.

## Installation

You need Python 3.10.11 to play this game. [Python 3.10.11](https://www.python.org/downloads/) to install Python.
This game will be run on Thonny version 4.1.7. [Thonny 4.1.7](https://thonny.org/) to install Thonny.

## Run the Program

```python
python game.py
```
